Question 11.1

Using the crime data set uscrime.txt from Questions 8.2, 9.1, and 10.1, build a regression model using:

  1. Stepwise regression
  2. Lasso
  3. Elastic net

For Parts 2 and 3, remember to scale the data first – otherwise, the regression coefficients will be on different scales and the constraint won’t have the desired effect. For Parts 2 and 3, use the glmnet function in R.

Notes on R:

  • For the elastic net model, what we called λ in the videos, glmnet calls “alpha”; you can get a range of results by varying alpha from 1 (lasso) to 0 (ridge regression) [and, of course, other values of alpha in between].
  • In a function call like glmnet(x,y,family=”mgaussian”,alpha=1) the predictors x need to be in R’s matrix format, rather than data frame format. You can convert a data frame to a matrix using as.matrix – for example, x <- as.matrix(data[,1:n-1])
  • Rather than specifying a value of T, glmnet returns models for a variety of values of T.

Stepwise regression - Manual Method

Function to calculate the AIC for each combination of predictors

#function to calculate AIC
calculate_aic<-function(predictors,primaryFormula) {
  aic_table<-NULL
  num_of_predict<-length(predictors)
  for (i in 1:num_of_predict) {
    formula<-as.formula(paste(primaryFormula, predictors[i], sep="+"))
    
    #get the aic and the predictor value after training a regression model
    aic_table<-rbind.data.frame(aic_table,data.frame("aic"=(glm(formula, data=training ))$aic,"predictor"=predictors[i]))
  
}

  return (aic_table)
}
  • Step 1: Regress each predictor on “Crime” separately, i.e. Crime ~ Po1, Crime ~ M, Crime ~ Ed etc and calculate teh AIC values returned. Pick the predictor with the lowest AIC value to start the step progression
  • Step 2: After selecting the best predictor from step 1 above, keep that predictor as constant in the model formula and add other predictors one by one. Repeat Step 1. ex: of Po1 was the predictor returned as the best predictor with low AIC from Step1, set the formula as “Crime ~ Po1” and fit a regressor for different predictors with Po1 being constant, i.e. Crime ~ Po1 + M , Crime ~ Po1 + Ed, Crime ~ Po1 + So etc …
  • Step 3: Repeat Step 1 and 2 and in each iteration select the best predictor that lowers the AIC value. If the AIC value stops improving break.
  • Step 4: fit a model with teh final set of predictors as provided by the iteration of Step 1 to Step 3
